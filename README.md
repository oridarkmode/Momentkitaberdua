# Undangan Digital – Glassmorphism Elegant

## Fitur
- Cover gate + custom nama tamu dari URL `?to=Nama+Tamu`
- Musik latar + mute
- Countdown
- Acara: Akad, Resepsi, Ngunduh Mantu + Maps + Simpan Calendar
- Galeri Foto + Video (modal)
- Kisah Cinta
- Kado Digital 3 opsi (copy/share)
- RSVP & Ucapan (localStorage) + opsi endpoint

## Cara Deploy GitHub Pages
Repo → Settings → Pages → Deploy from branch → main → /(root)

Contoh link tamu:
`https://username.github.io/repo/?to=Nama+Tamu`

## Ubah data
Edit file: `data/config.json`
